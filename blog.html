Question-1: What are the different ways to select an element in the DOM?

We can select element in DOM with some different ways. Here are, 1. getElementById , 2. getElementsByClassname , 3. getElementsByTagName, 4. querySelector and 5. querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Differences between innerHTML, innerText and textContent : 1. innerHTML return whole set of html content within an element. 2. innerText showes the visible text content of an element excluding hidden elements. 3. textContent showes text set of an element including hidden elements.

Question-3: What is event delegation in the DOM?

Add an EventListener in parent element with event parameter we can access all the element within the parent element by using event.target(). It helps to prevent add multiple eventListener in each element.

Question-4: What is event bubbling in the DOM?

When we add an event in a parent element it will be applicable in child element too. Suppose we target a div element, the parent of div is body will auto targeted, the parent of body element html tag also targeted and document too. By using event.stopPropagation() , We can stop event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

1.Create element: document.createElement('elementName').. 2.Add element: Suppose I have created a element div and I want to add it in section tag. I can add this through section.appendChild(div).. 3.Remove element: To remove a particular element firstly get it in a variable named elementName then use this formula parentElement.removeChild(elementName);
